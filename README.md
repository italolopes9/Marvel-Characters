#### Dependências

#### Tecnologias
- HTML
- CSS
- Javascript
- SCSS
- Bootstrap 4
- NodeJs
- Typescript
- Compodoc

------------

#### Comandos NodeJS
###### Baixar dependência
```shell
    npm install
```
###### Executar Aplicação
```shell
    npm dev || npm start
```

------------

#### Comandos Angular
###### Baixar dependência
```shell
    npm install
```
###### Executar Aplicação
```shell
   ng serve --open
```

------------

#### Comandos Documentação de Código (Compodoc)
###### Baixar dependência
```shell
    npm install
```
###### Executar Aplicação
```shell
   compodoc -s
```

###### Informações
```shell
   Acesse http://localhost:8080 para visualizar a documentação.
```
